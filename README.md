DesignPattern Project

This project contains implementations of three design patterns in C#:

1. Factory Pattern

Implements a database creation mechanism using a factory to produce different database types.
Classes located in the Factory folder:

IDatabase

MySQLDatabase

OracleDatabase

PostgreSQLDatabase

DatabaseFactory

DataAnalyticsEngine

2. Adapter Pattern

Adapts the HR employee data format to the BillingSystem format.
Classes located in the Adapter folder:

HRSystem

BillingSystem

EmployeeAdapter
